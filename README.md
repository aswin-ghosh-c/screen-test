# Screen Tester (Vue 3)

A Vue 3 application for testing screens (mobile and web). Features:
- Platform selection (Mobile/Web)
- Mobile: touch sensitivity, RGB color tests, dead pixel test
- Web: color tests, dead pixel test
- Each test has a dedicated view/component

## Usage
1. Run the app:
   ```sh
   npm run dev
   ```
2. Open the local URL in your browser.
3. Select your platform and desired test.

## Project Setup
- Built with [Vite](https://vitejs.dev/) and [Vue 3](https://vuejs.org/).

## Development
- Components are in `src/components/`
- Main logic in `src/App.vue`
